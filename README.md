# TraceLens Web

TraceLens Web is an installable Safari web app for tracing a selected picture over the live rear camera.

## Publish it

Camera access requires HTTPS. Upload the contents of this folder to any static website host, such as GitHub Pages, Cloudflare Pages, Netlify, or Vercel. No build command or server-side code is needed.

## Use it on iPhone

1. Open the published HTTPS address in Safari.
2. Tap **Start camera** and allow camera access.
3. Tap **Choose picture** and select an image.
4. Use two fingers to resize the image, one finger to move it, and the slider to change opacity.
5. To install it, tap Safari's **Share** button and choose **Add to Home Screen**.

The chosen picture is processed locally and is never uploaded by this app.
